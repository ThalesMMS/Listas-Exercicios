# Portal de colecao

Exemplo opcional para publicar varias copias independentes do Explainer a partir
de uma pagina raiz.

Uso recomendado:

1. Copie esta pasta para o nivel acima das suas pastas de explainers.
2. Edite os `href` dos cards em `index.html`.
3. Ajuste titulos, descricoes e contagens.

O portal nao importa arquivos de `Explainer-Template` nem de uma lista especifica.
Ele apenas aponta para pastas independentes e usa a mesma chave `ex-theme` para
sincronizar a preferencia claro/escuro quando os explainers tambem usam essa
chave.
