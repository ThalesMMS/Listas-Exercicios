# Compiladores - Lista A (interativa)

Resolucao comentada das 8 questoes da Lista A de Compiladores, baseada no
`Explainer-Template` como copia independente.

## Como abrir

Abra `index.html` no navegador. A lista funciona via `file://`, sem build e sem
dependencias.

- Hub: `index.html`
- Questao: `question.html?q=<id>`
- PDF original: `Compiladores-Lista-A.pdf`
